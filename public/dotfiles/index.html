Install

Want to use everything? This is how you install this.

Step 1
Clone the repository.
git clone --recurse-submodules https://gitlab.com/chtc/dotfiles.git $HOME/.dotfiles
Step 2
Install rcm.
Step 3
Install dependencies[3][4]
Step 4
Install the dotfiles with rcm.
RCRC=$HOME/.dotfiles/rcrc rcup
