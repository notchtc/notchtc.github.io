Install

Want to use everything? This is how you install this.

Step 1
Clone the repository.

Step 2
Install rcm.
Step 3
Install dependencies[2][3]
Step 4
Install the dotfiles with rcm.
RCRC=$HOME/.dotfiles/rcrc rcup
