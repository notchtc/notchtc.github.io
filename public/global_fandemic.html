Details

Splitted by using ffmpeg, python and stolen code from stackexchange.

Making of the cover art

  1. Downloaded 2 frames (the one with the global fandemic text and one before that) by using youtube-dl and ffmpeg.
  2. Removed most of the background from the frame with the text by using remove.bg.
  3. Removed the rest of the background in gimp.
  4. Cropped the frame without the text.
  5. Added the text onto the frames.
